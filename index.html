Projects

TaskBot for Slack

I was approached by a USC organization to develop a task manager for them. Since they were already using Slack, I decided to build a bot using Slack's API and Node.js on the backend. To store the task information, I used MongoDB to manage a NoSQL database. I also utilized the Express Framework to listen to requests sent by the Slack API.

Command Line Spoons

I created a basic game of spoons in C++ that is played on the command line. It uses Object-Oriented fundamentals in designing the cards and decks used to play the game. It also utilizes multithreading and several timers to mimic opponents taking a spoon. Read the instructions and try it out here!

MusIC Website

I am currently developing a website for an organization called Music Inspiring Community using React.js. The organization provides free music lessons for children in the LA area. The website will include forms for prospective students and parents to sign up for the program. It will also be updated regularly for parents to see what their children are learning. I plan to use Django to handle the forms. See my progress here!
